# MoodTunes

An engaging web app that generates custom music based on your mood using AI. Simply move your cursor across the colorful mood board, and watch as the AI creates unique melodies matching your emotional state in real-time. Share your favorite tunes with friends!

Features:
- Interactive mood board with smooth color transitions
- Real-time AI music generation
- Save and share favorite melodies
- No musical knowledge required
- Works on desktop and mobile

Tech Stack:
- ToneJS for audio synthesis
- Web Audio API
- HTML5 Canvas
- Vanilla JavaScript
